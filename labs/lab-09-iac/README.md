# Lab 09 — Infrastructure as Code

## Objectif
Implémenter un mini-moteur IaC simplifié en TypeScript.

## Exercices
1. Modéliser des ressources et dépendances
2. Calculer le plan de changements (diff)
3. Résoudre l'ordonnancement (graphe de dépendances)
4. Simuler l'application et la destruction

## Lancer
```bash
npx tsx labs/lab-09-iac/exercise.ts
npx tsx labs/lab-09-iac/solution.ts
```
