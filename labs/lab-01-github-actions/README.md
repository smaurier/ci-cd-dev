# Lab 01 — GitHub Actions Fondamentaux

## Objectif
Implémenter un parseur et validateur de workflows GitHub Actions.

## Exercices
1. Parser la structure d'un workflow YAML (triggers, jobs, steps)
2. Valider les triggers supportés
3. Détecter les dépendances entre jobs (needs)
4. Vérifier la cohérence d'un workflow (jobs référencés existent)

## Lancer
```bash
npx tsx labs/lab-01-github-actions/exercise.ts
npx tsx labs/lab-01-github-actions/solution.ts
```
