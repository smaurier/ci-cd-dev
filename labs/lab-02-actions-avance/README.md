# Lab 02 — GitHub Actions Avancé

## Objectif
Implémenter un système de build matrix et de workflows réutilisables.

## Exercices
1. Générer une build matrix (OS × Node version)
2. Filtrer et exclure des combinaisons
3. Gérer les outputs entre jobs
4. Simuler des workflows réutilisables avec inputs/outputs

## Lancer
```bash
npx tsx labs/lab-02-actions-avance/exercise.ts
npx tsx labs/lab-02-actions-avance/solution.ts
```
