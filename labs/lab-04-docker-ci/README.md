# Lab 04 — Docker dans le CI

## Objectif
Implémenter un système d'analyse et d'optimisation de Dockerfiles.

## Exercices
1. Parser un Dockerfile multi-stage
2. Analyser les layers et optimisations
3. Générer des tags sémantiques pour les images
4. Scanner les vulnérabilités
