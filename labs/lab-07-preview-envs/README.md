# Lab 07 — Preview Environments

## Objectif
Implémenter la gestion d'environnements éphémères pour les pull requests.

## Exercices
1. Gérer le lifecycle des preview environments
2. Générer les URLs et configurations
3. Nettoyer les environnements obsolètes
4. Gérer les bases de données éphémères
