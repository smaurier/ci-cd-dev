# Lab 08 — Sécurité des Pipelines CI

## Objectif
Implémenter des contrôles de sécurité pour les pipelines CI/CD.

## Exercices
1. Vérifier les permissions d'un workflow
2. Détecter les injections de script dans les workflows
3. Valider l'épinglage des actions par hash
4. Auditer la configuration des secrets

```bash
npx tsx labs/lab-08-securite-ci/exercise.ts
npx tsx labs/lab-08-securite-ci/solution.ts
```
