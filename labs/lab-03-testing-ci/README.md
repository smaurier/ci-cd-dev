# Lab 03 — Testing dans le CI

## Objectif
Implémenter un système de gestion des tests dans un pipeline CI.

## Exercices
1. Configurer le sharding des tests
2. Calculer les quality gates (couverture, seuils)
3. Détecter les tests flaky
4. Agréger les résultats de tests parallèles

## Lancer
```bash
npx tsx labs/lab-03-testing-ci/exercise.ts
npx tsx labs/lab-03-testing-ci/solution.ts
```
