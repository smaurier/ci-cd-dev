# Lab 06 — Stratégies de Déploiement

## Objectif
Implémenter et simuler différentes stratégies de déploiement.

## Exercices
1. Simuler un déploiement rolling update
2. Implémenter un blue-green switch
3. Simuler un canary progressif avec métriques
4. Décider du rollback automatique
