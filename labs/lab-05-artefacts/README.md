# Lab 05 — Artefacts et Registries

## Objectif
Implémenter un système de gestion d'artefacts et de versioning sémantique.

## Exercices
1. Gestion d'artefacts (upload, download, rétention)
2. Versioning sémantique (bump, comparaison)
3. Publication de packages avec scopes
4. Génération de changelogs depuis les commits

## Lancer
```bash
npx tsx labs/lab-05-artefacts/exercise.ts
npx tsx labs/lab-05-artefacts/solution.ts
```
